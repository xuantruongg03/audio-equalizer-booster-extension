# Icon Generation

## How to generate icons

1. Open `generate-icons.html` in a browser
2. Click the download buttons for each size
3. Save the PNG files as:
   - `icon16.png`
   - `icon32.png`
   - `icon48.png`
   - `icon128.png`

## Icon Design

The icon features:
- Gradient background (blue to purple)
- Rounded corners
- White equalizer bars representing audio levels
- Clean, modern aesthetic matching the extension UI

## Required Sizes

Chrome extensions require icons in the following sizes:
- 16x16 - Toolbar icon
- 32x32 - Windows computers
- 48x48 - Extension management page
- 128x128 - Chrome Web Store
